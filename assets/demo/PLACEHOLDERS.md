# Demo media placeholders

Drop screenshots, GIFs, and screen-recording videos here. The top-level `README.md` links to the following paths — replace them with the real files (keep the filenames stable or update the README).

| File                           | Purpose                                        |
|--------------------------------|------------------------------------------------|
| `assets/demo/overview.mp4`     | 30-60s "what is Actionow" overview video      |
| `assets/demo/overview.gif`     | Short looped GIF for README hero section       |
| `assets/demo/screenshot-chat.png`       | Agent SSE chat UI                     |
| `assets/demo/screenshot-canvas.png`     | Infinite canvas / storyboard         |
| `assets/demo/screenshot-mission.png`    | Mission (long-running task) progress |
| `assets/demo/screenshot-skills.png`     | Skill marketplace / imports          |
| `assets/demo/architecture.png`          | Architecture diagram (optional)      |

Recommended sizes:

- Hero GIF: 1200×675, ≤ 5 MB
- Screenshots: 1920×1080 PNG or 1440×900 Retina
- Videos: MP4 (H.264 + AAC), ≤ 20 MB for GitHub preview
